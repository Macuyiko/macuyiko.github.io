List of Apps (2018)

It’s time to setup a new laptop, so once again I’m jotting down a list of apps I normally install right away, slightly updated again.

Web

  • Chrome
  • Dropbox
  • Google Drive
  • FileZilla
  • qBittorrent

Skype is avoided, use web version once again.

General

  • Microsoft Office
  • 7-Zip
  • Everything to locate files and folders by name instantly.
  • f.lux
  • Paint.NET
  • Sumatra PDF and Adobe Reader for those annoying edge cases where Sumatra doesn’t work.
  • VirtualBox
  • Spotify
  • VLC
  • Steam
  • GOG Galaxy
  • Fonts: Open Sans especially.

Cmder is out, use PowerShell or Ubuntu from the Windows store.

Data Science

  • Anaconda Python Distribution
  • GraphViz
  • R and RStudio

Coding and Editing

  • Eclipse
  • GitHub Desktop
  • LyX
  • Sublime Text
  • VSCode: excellent and rapidly replacing every other code editor.
  • WinEdt
